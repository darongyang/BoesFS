# `fn unset`

```c
$ fn unset

MANAGEMENT COMMAND
  fn unset -   Unset elements of a created object
               
USAGE
  fn [global options] unset [command options] <subcommand>
    
DESCRIPTION
  This command unsets elements ('configurations') for a created object ('app', 'function' or 'context').
    
SUBCOMMANDS
  config, config, cf      Unset configurations for apps and functions
  context, ctx            Unset current-context
  help, h                 Shows a list of commands or help for one command
                      
COMMAND OPTIONS
  --help, -h  show help

FURTHER HELP:  See 'fn unset <subcommand> --help' for more information about a command.
```
